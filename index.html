MacRuby MP3 Audio Player

This is a simple MP3 Audio Player using MacRuby and hotcocoa.

Prerequisites

  • MacRuby >= 0.6
  • hotcocoa

Running from source

$ macrake run

or

$ macruby lib/application.rb

Building WMPlayer.app

$ macrake deploy
